Nous sommes désolé, mais le problème vient probablement de vous ( sans être catégorique, prudence est mère de sûreté ;) )

Bon, nous allons essayer de vous aider au mieux. Voici une liste qui contiendra probablement des informations utiles :

      Si vous avez rencontré un problème lors d'une recherche de produit :
    • Le produit cherché n'est pas contenu en base de données
    • Vous n'avez pas spécifié de valeur dans le champ de recherche

      Si vous avez rencontré un problème lors de votre connexion ou votre tentative de création de compte :
    • Le compte que vous essayez de créer existe déjà
    • Le compte que vous essayez de créer possède une adresse mail invalide
    • Le compte auquel vous voulez vous connecter n'existe pas
    • Vous n'avez pas spécifié de valeur pour les champs UserName et eMail

      Si vous rencontrez un problème lors d'une tentative d'ajout d'un produit en favoris ou de la consultation de vos favoris :
    • Vous tentez d'accéder à vos favoris alors que vous n'êtes pas connecté
    • Vous tentez d'ajouter un produit à vos favoris alors que vous n'êtes pas connecté
    • Le produit que vous voulez ajouter se trouve déjà dans vos favoris

Il est possible que votre problème ne soit pas lié aux cas cités ci-dessus. Dans ce cas, veuillez nous contacter par mail pour faire remonter votre problème à nos équipes.
